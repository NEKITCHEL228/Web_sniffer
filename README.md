# Web_Sniffer (Кроссплатформенный сетевой сниффер)

## Суть проекта
Данное приложение предназначено для захвата и анализа сетевого трафика в реальном времени. Разработано с использованием паттернов проектирования (Singleton, Observer, Facade, Strategy). 

## Инструкция по сборке (Out-of-source build)
Для сборки проекта используется система CMake. Выполните следующие команды в терминале:

```bash
cmake -S . -B build
cmake --build build
```

## Пример запуска
Перейдите в директорию сборки и запустите исполняемый файл:
```bash
cd build
./Web_Sniffer
```

## Список использованных сторонних библиотек
* **Qt6 (Widgets)** — реализация кроссплатформенного графического интерфейса (GUI) и паттерна Наблюдатель (Signals & Slots).
* **libpcap / Npcap** — низкоуровневая библиотека для захвата сетевых пакетов на сетевом интерфейсе устройства.
